# Will Your Career Survive AI by 2030?

This project analyzes 3,000 job profiles to evaluate automation risk by 2030.

It explores:
- Automation probability distribution
- Salary vs risk
- Experience vs risk
- Technological growth tradeoff
- A custom Future Stability Score
